# Colombia 5.0 - Experiencia Académica

## Descripción

Este proyecto consiste en un sitio web desarrollado como evidencia académica para documentar la participación en el evento Colombia 5.0 realizado en Bogotá.

La página presenta información sobre las conferencias asistidas, los conceptos tecnológicos aprendidos, una galería fotográfica del evento y una reflexión personal sobre la importancia de la innovación y la transformación digital.

## Objetivos

- Documentar la experiencia vivida durante Colombia 5.0.
- Presentar los conocimientos adquiridos en las conferencias.
- Fortalecer habilidades de desarrollo web.
- Aplicar buenas prácticas de diseño y experiencia de usuario.

## Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript
- GitHub Pages

## Características

- Diseño responsive.
- Navegación fluida.
- Galería interactiva.
- Lightbox para visualización de imágenes.
- Glosario tecnológico.
- Animaciones y efectos visuales.
- Compatible con dispositivos móviles y escritorio.
